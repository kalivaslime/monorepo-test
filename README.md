## Comands

project init

```sh
pnpm init
```

## Install root deps

typescipt

```sh
pnpm add -Dw typescript
pnpm tsc --init
```

## React

```sh
pnpm add -Dw react react-dom
pnpm add -Dw @types/react @types/react-dom
```

## Parcel

```sh
pnpm add -Dw parcel
```

## Add package to root

```sh
pnpm add -Dw package-name
```

## Add package to workspace

```sh
pnpm --filter workspace-name add package-name
```
